# 💱 Currency Data Scraper

## 🇧🇷 Descrição
Aplicação desenvolvida em Python para coleta e processamento de dados de moedas internacionais utilizando técnicas de web scraping.

O **Currency Data Scraper** permite extrair informações diretamente de fontes externas, simulando um fluxo real de coleta e tratamento de dados financeiros.

---

## 🇺🇸 Description
A Python application designed to collect and process international currency data using web scraping techniques.

**Currency Data Scraper** retrieves currency information from external sources, simulating a real-world financial data extraction workflow.

---

## 🚀 Funcionalidades | Features
- Coleta de dados de moedas via HTTP requests  
- Parsing de HTML com BeautifulSoup  
- Estruturação de dados em listas e dicionários  
- Interface interativa via terminal (CLI)  
- Validação de entrada do usuário  
- Tratamento de erros  

---

## 🧠 Tecnologias utilizadas | Tech Stack
- Python  
- Requests  
- BeautifulSoup  

---

## 📦 Pré-requisitos | Requirements
Antes de executar, instale as dependências:
```bash
pip install -r requirements.txt
```

---

## ▶️ Como executar | How to run
```bash
git clone https://github.com/jomarferreira/currency-data-scraper.git
cd currency-data-scraper
pip install -r requirements.txt
python main.py
```
